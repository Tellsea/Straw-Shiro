# Straw-Shiro

### 项目简介

基于 springboot、shiro 实现的后台管理系统模板，主要可以用于线上项目、外包项目、企业项目的生产使用，其中使用的相关组件均为开源组件，并未涉及授权购买等相关问题，可以放心使用。使用的相关技术均为市面上通用的技术，不会存在有问题，没答案的现象。
